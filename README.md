React Blog App:
A simple blog application built using React with JSON Server as a mock backend.

Technologies Used:
React
JSON Server

How to Run the Project
1. Install Dependencies
npm install

2. Start JSON Server
npx json-server --watch data/db.json --port 8000


JSON Server runs on http://localhost:8000

3. Start React App
npm start

App runs on http://localhost:3000

Important Note

JSON Server must be running to view blog data
Blog data is stored in data/db.json
